# Crops-Ciencia-De-Datos
El conjunto de datos de recomendación de cultivos es un conjunto de datos estructurado diseñado para ayudar en la toma de decisiones agrícolas mediante la recomendación de cultivos adecuados en función de diversas condiciones ambientales y del suelo.


Análisis y Predicción de Cultivos a partir de Variables Ambientales
Objetivo General:
Clasificar el tipo de cultivo con el fin de lograr la identificación del más adecuado según las características del suelo, las condiciones climáticas y la disponibilidad de nutrientes.
Objetivos Específicos:
•	Describir las variables presentes en el conjunto de datos (Temperatura, Humidity, pH, Rainfall) para comprender sus características y rangos.
•	Seleccionar el modelo de aprendizaje óptimo para predecir la categoría del cultivo correspondiente.
•	Formular conclusiones basadas en el análisis realizado para apoyar la selección del cultivo más adecuado según las condiciones ambientales.
